# 🖼️ Outil de Stéganographie BMP - Win32 API

### Description
Application Windows native permettant de cacher et extraire des messages texte dans des images BMP via la méthode LSB.

### Fonctionnalités
- Chargement / affichage d'une image BMP
- Insertion d’un message texte dans les pixels
- Extraction d’un message caché
- Sauvegarde de l’image modifiée

### Technologies
- C / C++ (Visual Studio 2022)
- API Win32 + GDI
- BMP 24/32 bits

### Utilisation
1. Fichier → Ouvrir
2. Stéganographie → Intégrer un message
3. Fichier → Enregistrer sous...
4. Stéganographie → Extraire un message
